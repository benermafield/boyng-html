# Sound Files

This directory should contain the following sound effect files:

- **bounce.wav** - Ball collision sound (short, punchy sound)
- **jump.wav** - Jump sound (slightly higher pitched than bounce)

## Where to Get Sounds

### Option 1: Free Sound Libraries
- [Freesound.org](https://freesound.org/) - Search for "bounce" and "jump"
- [OpenGameArt.org](https://opengameart.org/) - Game sound effects
- [Mixkit.co](https://mixkit.co/free-sound-effects/) - Free sound effects

### Option 2: Generate with Web Audio API
Use online tools like [SFXR](https://sfxr.me/) or [Bfxr](https://www.bfxr.net/) to generate simple game sounds.

### Option 3: Create in Audacity
1. Generate a simple tone (Generate → Tone)
2. Apply fade in/out effects
3. Export as WAV file

## Current Status
The game will load and play even without these files. Phaser handles missing audio gracefully. Add actual sound files here to enable audio feedback.
